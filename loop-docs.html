Loops with Strings

üìù What is a String?

A string is just text inside quotes. Like a line of letters:

word = "Apple"

This string has 5 letters: A p p l e

‚úã The Hard Way (Without Loop)

Printing each letter separately:

word = "Apple"

print("A")
print("p")
print("p")
print("l")
print("e")

üòÖ Too much typing! What if the word had 20 letters?

üîÅ The Smart Way (With For Loop)

Let Python do the work for you:

word = "Apple"

for letter in word:
    print(letter)

How it works:

  1. First time ‚Üí letter = "A", prints A
  2. Second time ‚Üí letter = "p", prints p
  3. Third time ‚Üí letter = "p", prints p
  4. Fourth time ‚Üí letter = "l", prints l
  5. Fifth time ‚Üí letter = "e", prints e

üòä Fun Example with Emojis

Loops with Lists

üß∫ What is a List?

A list is like a basket that holds many items together:

fruits = ["Apple", "Banana", "Cherry"]

Each fruit has a position number (index):

Apple
Index: 0
Banana
Index: 1
Cherry
Index: 2

üîÅ Looping Through a List

fruits = ["Apple", "Banana", "Cherry"]

for fruit in fruits:
    print(fruit)

How it works:

  1. First time ‚Üí fruit = "Apple", prints Apple
  2. Second time ‚Üí fruit = "Banana", prints Banana
  3. Third time ‚Üí fruit = "Cherry", prints Cherry

üåà Fun Example with Colors

colors = ["Red", "Green", "Blue"]

for color in colors:
    print(color + " is my favorite!")

Loops with Tuples

ü•° What is a Tuple?

A tuple is like a sealed box - you can't change the items inside!

animals = ("Dog", "Cat", "Rabbit")

Key Difference: Lists can change (add/remove items), but tuples cannot!

üîÅ Looping Through a Tuple

Works just like with lists:

animals = ("Dog", "Cat", "Rabbit")

for animal in animals:
    print(animal + " lives here")

Output:

Dog lives here
Cat lives here
Rabbit lives here

üéØ What We Learned

Strings

  • Loop through each letter
  • Use for letter in word

Lists

  • Loop through each item
  • Use for item in my_list

Tuples

  • Loop through each item
  • Just like lists, but can't change items
